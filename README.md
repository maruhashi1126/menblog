# **men,s blog**

# **概要**

麺料理の情報共有のできるブログサイトを作りました。


# **使い方**

自分の食べた料理を写真を載せて別ユーザーと情報を共有できるページを作成いたしました。

気に入ったお店があった場合、ユーザー登録したユーザーはいいねをすることができます。

また、ユーザー登録していないユーザーはお店の情報や、コメントを送ることができます。
